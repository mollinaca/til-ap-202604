# ソフトウェア品質とレビュー

システム/ソフトウェア製品の品質規格(JIS X 25010)  
JIS X 25010(ISO/IEC 25010) → 利用時の品質と製品品質  

利用時の品質

* 有効性
* 効率性
* 満足性
* リスク回避性
* 利用状況網羅性

製品品質

* 機能適合性
* 性能効率性
* 互換性
* 使用性
* 信頼性
* セキュリティ
* 保守性
* 移植性

レビュー

* ウォークスルー → 開発担当者が説明する
* インスペクション → モデレータが進行する
