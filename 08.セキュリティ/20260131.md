# 無線LANのセキュリティ

* MACアドレスとフィルタリング
* ESSID, SSID
  * ANY 接続拒否：SSIDが一致しないと接続しない
  * ステルス機能：ビーコン信号に自身のSSID名を乗せない

暗号化方式

* WEP : RC4、古い
* WPA : TKIP, WPA2 までのつなぎ
* WPA2 : AES(CCMP)
* WPA3 : SAE, CNSA

# FW

* DMZ
* パケットフィルタリング
* WAF

# 侵入検知・侵入防止

* IDS(侵入検知)
* IPS(侵入防止)

それぞれ、シグネチャ型とアノマリ型がある
 → false positive, false nagative の概念
