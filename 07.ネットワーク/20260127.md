# メディアアクセス制御

* MAC アドレス（L2）
* CSMA/CD方式：有線LANにおける衝突検知、イーサネット時代の技術で絶滅
* トークンパッシング方式：送信権を管理する方式、絶滅
* CSMA/CA方式：衝突を避ける、現代のWi-Fiでも使われている、
* TDMA：時間をスライスしてわける、予約制、5G/6G、衛星通信などに使われている
現代の有線LAN環境はスイッチングにより送信網と受信網が2重化されているので、そもそも衝突が発生しない

# 無線LAN

CSMA/CA：広い空間を使って衝突をさける、誰かのデータがあったら、終わるのを待ち、さらに少し待機してからデータを送る、衝突を極力回避する

# LAN に関する規格

IEEE 802

* リンクアグリゲーション IEEE 802.3ad：複数のポートを束ねて高速化、冗長化する
* PoE IEEE 802.3af:LANケーブルを通じて危機への給電を行う
* スパニングツリープロトコル IEEE 802.1D:ループを防止しながら冗長性を高める、ストームを防ぐ
