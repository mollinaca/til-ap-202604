# 確率

## 基本的な話

確率、独立、排他、従属など基本的な話

## ベイズの定理

ベイズの定理、結果から原因の確率を推定する

```
P = (Aが起こりNが起こる確率)/Nが起こる確率
```

これにより、Nの原因がAだったといえるかどうかの確率を求められる。
"結果から原因の確率を求める" を繰り返すことで、より確率を上げられる → 機械学習やAI、そのほか社会的な様々な要因分析手法として使われる。
ベイズ統計、ベイジアンフィルタなど。

# 確率分布

## 正規分布（ガウス分布）

連続型の確率分布、平均を中心とした左右対称の山形。平均と標準偏差。
