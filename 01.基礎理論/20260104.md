## 待ち行列

M1/M2/1(∞) or M1/M2/S  

* M1 → 単位時間あたりのトランザクション到着数、ポアソン分布。到着間隔は指数分布。  
* M2 → サービス処理にかかる時間、指数分布  
* 1 or S → 窓口が一つ、もしくは複数  
* ∞ → 行列は十分に長く、キューがからになることはない  

要素

* 平均到着率 → λで表記する、単位時間あたりに到着するトランザクション数（M1）
* 平均到着間隔 → 1/λ
* 平均サービス率 → 単位時間あたりにサービス可能なトランザクション数、μで表記する
* 平均サービス時間 → サービスを受ける時間の平均、 1/μ
* 利用率 → 単位時間あたりの窓口利用率、ρ で表記する、ρ = λ/μ

よくある計算

* 平均待ち時間 = ρ / (1-ρ) * 平均サービス時間
* 平均応答時間 = 平均待ち時間 + 平均サービス時間
