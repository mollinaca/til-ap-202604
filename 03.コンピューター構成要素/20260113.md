# 高信頼システム

カタカナ語いっぱい  
フォールトトレランス  
フォールトアボイダンス  
フェールセーフ  
フェールソフト  
フールプルーフ  

デュアルシステム → Act/Act で相互に処理チェックすることで信頼性を高める、片方が障害のときは縮退する  
デュプレックスシステム → Act/Standby 、障害のときは standby へ切り替え  
standby 方式 → コールドスタンバイ、ウォームスタンバイ、ホットスタンバイ  
ホットスタンバイではハートビートの使い方（Act→Standby への定期通知）が聞かれる
