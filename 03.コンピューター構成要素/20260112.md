# NAS と SAN

NAS:Network Attached Storage ディスク装置を直接LANに背t図億する、SMB, CIFS, NFS
SAN:Storage Area Network ストレージを通常のLANとは別の高速ネットワークで構成する、FC、iSCSI、FCoE、サーバやLANを介さない専用ネットワーク
NFS:Network FIぇSystem、RPC、UNIXで利用されるファイル共有システム

# RAID

RAID0:ストライピング、速度上昇、どれか一台でも壊れると全体が死ぬ
RAID1:ミラーリング、冗長性、容量半分
RAID2:RAID0にエラー訂正符号（ハミング符号）を追加、障害児に復旧できるようになった
RAID3,RAID4:RAID0にパリティディスクを追加、RAID3はビット単位、RAID4はブロック単位でストライピングする
RAID5:RAID4は書き込みの際にパリティディスクへアクセスが集中するので、データとパリティを複数のディスク間で復元する

RAIDの稼働率
RAID0 非冗長構成、稼働率をνとすると、2台構成のとき全体の稼働率はν^2
RAID1 冗長構成、2台のうちどちらか1台だけでも稼働していれば全体としては動く、
2台が同時に動く確率は ν^2、で、どちらか片方だけ動く確率は 2C1 *ν* (1-ν)^(2-1) = 2 *ν* (1-ν) = 2ν(1-ν)
 = ν^2 + 2ν(1-ν) = ν^2 + 2ν - 2ν^2 = 2ν - ν^2

RAID01 : 2ν^2 - ν^4
